# Documentation Site

A single-file documentation website for this repository — ReadTheDocs-style recipes,
benchmark results, hardware reference (with dual-NUMA topology infographic), and
deep dives (agentic benchmark, long-context, GKE Inference Gateway, NCCL/iperf3).

## View locally
Open `docs/index.html` in any browser. No build step, no dependencies.

## Publish with GitHub Pages
Repo Settings → Pages → Source: "Deploy from a branch" → Branch: `main`, folder `/docs`.
The site will be served at `https://<user>.github.io/sglang-rtx-pro-6000/`.
