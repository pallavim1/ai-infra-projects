# AI Infrastructure Projects

This repository hosts optimization guides, benchmark scripts, and cluster deployment templates for various GPU hardware environments.

## Repository Layout
All serving and inference benchmarking resources are grouped under the [Inference/](file:///usr/local/google/home/pallaviam/AI%20Infra%20Projects/ai-infra-projects/Inference) directory:

- [Inference/sglang-rtx-pro-6000/](file:///usr/local/google/home/pallaviam/AI%20Infra%20Projects/ai-infra-projects/Inference/sglang-rtx-pro-6000)
  Optimizations, models, and GKE cluster configurations tailored for RTX Pro 6000 (G4) GPUs.
  
- [Inference/sglang-h100/](file:///usr/local/google/home/pallaviam/AI%20Infra%20Projects/ai-infra-projects/Inference/sglang-h100)
  Multi-node serving configurations, GPUDirect TCPX network setup guides, and GKE cluster creation templates optimized for H100 (A3 Mega) GPUs.

- [Inference/sglang-h200/](file:///usr/local/google/home/pallaviam/AI%20Infra%20Projects/ai-infra-projects/Inference/sglang-h200)
  Placeholder directory for H200 GPU SGLang inference benchmarks.
